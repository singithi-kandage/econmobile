Clarification regarding
required VMs		 Sep 20, 2017 11:30 am
Hi everyone,

I've been getting a few questions about what VMs you require so I thought I would post here to ensure everyone is clear.

You have a set of VMs (2 at this point) that we have started using for in-class work. Those VMs should be used for in-class work only. I have also been posting and will continue to post practice exercises. You will need a separate set of VMs to complete those since you will either not be able to complete both in-class work and practice exercises on the same VMs, or if you can it will likely cause a lot of confusion.

You do not have to bring your practice exercise VMs to class. If you have room on your USB storage, you may want to bring them along. There will be some days that I will provide time to start on practice exercises or get caught up on previous exercises you have not finished.
